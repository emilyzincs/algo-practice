# General Implementation Requirements
- Must be fully contained in the [practice file](main-loop).
- 

## Python
- A method named ```solve``` must implement the algorithm, and it must belong to a top-level class named ```solution```.

## Java
- A public static method named ```solve``` must implement the algorithm, and it must belong to a top-level public class named ```solution```.

# Class-specific Implementation Requirements
This project supports 

## Python
- ListNode
  - class with same name that has val field of the required type and next field of type Optional[ListNode]
- TreeNode
  - class with same name that has val field of the required type and left and right fields type Optional[TreeNode]
